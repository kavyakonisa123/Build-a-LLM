# Building an LLM from Scratch

This repository documents my learning of building a **Large Language Model (LLM) from scratch**, inspired by the excellent resource:
*Build a Large Language Model from Scratch* by Sebastian Raschka ([https://sebastianraschka.com/](https://sebastianraschka.com/)).

## Overview

This project is a hands-on learning experience where I progressively implement the core components of modern LLMs. The goal is to deeply understand how these models work under the hood by building them step by step.

### 1. PyTorch Fundamentals

* Understanding tensors, operations, and autograd

### 2. Tokenization (First Step)

* Converting raw text into tokens
* Exploring different tokenization strategies
* Preparing inputs for language models

### 3. Attention Mechanisms

* Simplified self-attention
* self-attention
* Causal (masked) attention
* Building up to **multi-head attention**

### 4. LLM Architecture

* Understanding transformer architecture
* Implementing a **GPT-style model from scratch**

### 5. Training the Model

* Pre-training on unlabeled text data
* Learning representations from scratch

### 6. Fine-Tuning

* Adapting the model for downstream tasks
* Improving performance on specific use cases

## Goal

This project is both:

* A **learning exercise** to understand LLMs in depth
* A **practical implementation** of core concepts from scratch

## Tech Stack

* Python
* PyTorch

## Inspiration

This work is heavily inspired by Sebastian Raschka’s book and tutorials, which provide a clear and practical approach to understanding and building LLMs.
